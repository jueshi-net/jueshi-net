## v1.20.19 — 会员等级 / 成长值 / 勋章系统

**目标**：建立用户等级体系，激励活跃度和付费转化

### 核心功能
- [ ] 等级定义（Lv.1 ~ Lv.5）
- [ ] 成长值计算规则（签到/发文/点评/付费）
- [ ] 勋章定义与展示（后台可配置）
- [ ] 前台用户主页展示等级和勋章
- [ ] 后台用户管理增加等级/成长值查看

### DB 变更
- `user_levels` 表：等级定义、所需成长值、权益
- `user_badges` 表：勋章定义
- `user_badge_awards` 表：用户获得的勋章
- `growth_logs` 表：成长值流水

### 风险
- 成长值计算涉及多模块联动（签到/点评/付费），需定义清晰的触发点
- 现有 `users.memberUntil` 字段需与等级系统对齐

## v1.20.22 — DIY HTML 小工具沙盒

**目标**：用户可编写/运行简单 HTML/JS 小工具

### 核心功能
- [ ] 沙盒编辑器（textarea + 预览 iframe）
- [ ] 预置模板（汇率换算、单位换算、倒计时等）
- [ ] 用户保存自定义工具
- [ ] 前台工具中心增加"我的工具"入口
- [ ] 安全隔离（iframe sandbox）

### DB 变更
- `user_tools` 表：用户自定义工具（HTML 内容、模板 ID、创建者）

### 风险
- ⚠️ XSS 安全：iframe 必须使用 `sandbox` 属性，禁止 `allow-scripts` 以外的权限
- ⚠️ 存储安全：用户 HTML 内容需过滤危险标签（script 除外，因沙盒需要）
- 建议放到 v1.20.24 或更后，安全审查优先

## v1.20.23 — 专题模板批量生产与视频 SEO 增强

**目标**：提高专题生产效率，增强 YouTube 视频的 SEO 价值

### 核心功能
- [ ] 专题模板系统（JSON schema 定义专题结构）

- [ ] 专题封面图自动生成（OG Image）

### DB 变更
- `topic_templates` 表：模板定义
- 现有 `topics` 表新增：

  - `templateId` 字段（String?）

### 风险

- OG Image 生成需要 headless browser 或 API 服务

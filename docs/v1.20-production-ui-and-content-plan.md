## 二、单据模板真实商务化升级

### 2.1 现状问题（v1.19.2 结论）
- 当前 8 种模板类型在**字段上确实不同**（唛头/集运/库位等各有专属字段）
- 但 6 种视觉风格**主要是颜色差异**，布局差异不明显
- 模板选择器在 UI 中存在，但可能不够醒目

### 2.2 升级方向
- 每种模板类型设计独立的**商务级排版布局**
  - 通用外箱唛头：标准外贸格式，含主唛/侧唛/箱号区域
  - 集运入库标签：仓库操作格式，大字突出账号和单号
  - 集运合箱标签：合箱专用格式，突出合箱编号和原包裹数
  - 仓库库位标签：货架标签格式，突出库位编号
  - 国际包裹面单：运单风格，寄件人/收件人分区
  - FBA 外箱信息贴：Amazon 风格参考格式
  - 托盘标签：大尺寸托盘标签格式
  - 提示标签：警示标签格式，图标+大字
- 增加真实商务模板的视觉参考
- 支持自定义公司 Logo（member 功能已有，需提升体验）
- 支持多套公司信息切换（member 功能已有，需提升体验）

### 2.3 优先级
- P0: 唛头模板商务化（最常用）
- P1: 集运入库/合箱标签
- P2: 其他模板

## 三、资源库/文章/网盘分享 CMS 化

### 3.1 资源库 CMS 化
- **现状**: DB 中有 80 条数据，后台有页面但需验证 CRUD 完整性
- **目标**: 后台可完整管理资源（增/删/改/排序/分类/标签）
- **字段**: 名称/URL/描述/分类/标签/来源类型/使用说明/免责声明/排序/启用状态
- **优先级**: P0

### 3.2 文章系统增强
- **现状**: 后台 Markdown 编辑器已可用
- **目标**:
  - 增加封面图上传
  - 增加分类/标签关联
  - 增加文章关联工具功能
  - 增加 SEO 标题/描述编辑
  - 增加定时发布
- **优先级**: P1

### 3.3 网盘分享模块
- **现状**: 未实现
- **目标**: 后台可管理网盘分享链接（名称/链接/提取码/分类/有效期/说明）
- **前台**: 独立的网盘分享页面
- **优先级**: P1

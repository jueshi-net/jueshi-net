# v1.18 轻量 UGC + 工具短评 + 排行榜 — 开发规划

## 版本信息

- **目标版本**: v1.18-stable
- **前置版本**: v1.17.1-stable
- **定位**: 轻量社区感和积分互动，不做重论坛

## 目标

给现有工具和场景包增加「短评 / 评分 / 排行榜」能力：
1. 用户可以对工具写 10-140 字短评 + 1-5 星评分
2. 提交短评获得 +10 积分（每日上限 30 分）
3. 管理员可审核/隐藏短评
4. 工具排行榜页面（Top 20）
5. 场景工具页展示评分和短评摘要
6. 首页展示本周热门工具 Top 5

## 数据模型

### ToolReview（已存在于 schema）

```prisma
model ToolReview {
  id            String   @id @default(cuid())
  userId        String
  toolKey       String
  rating        Int            // 1-5
  content       String         // 10-140 字
  status        String   @default("pending")  // pending/approved/rejected/hidden
  pointsAwarded Boolean  @default(false)
  ipHash        String?
  userAgent     String?
  createdAt     DateTime @default(now())
  updatedAt     DateTime @updatedAt

  user User @relation(fields: [userId], references: [id], onDelete: Cascade)

  @@unique([userId, toolKey])
  @@index([toolKey])
  @@index([status])
  @@index([createdAt])
  @@map("tool_reviews")
}
```

约束：
- 每个用户对同一个工具只能写一条短评
- 可以编辑，但 approved 编辑后重新变 pending

## API 列表

| 方法 | 路径 | 权限 | 说明 |
|------|------|------|------|
| GET | /api/tools/reviews?toolKey=xxx | 公开 | 获取工具 approved 短评 + 平均分 |
| POST | /api/tools/reviews | 登录 | 提交短评 |
| PATCH | /api/tools/reviews/[id] | 登录（本人） | 编辑短评 |
| GET | /api/tools/rankings | 公开 | 工具排行榜 Top 20 |

| PATCH | /api/admin/tool-reviews/[id] | Admin | 审核通过/拒绝/隐藏 |

## 积分规则

- 提交一条有效短评：+10 积分
- 每天最多 30 分（3 条）
- 同一个 toolKey 只奖励一次
- 写入 PointLedger: type=tool_review
- 防刷：内容长度 10-140 字、禁止链接、禁止纯符号

## 防刷机制

1. userId + toolKey unique 约束
2. 每日积分上限 30 分
3. Content 校验：
   - 10-140 字
   - 禁止 http/https 链接
   - 禁止纯 emoji/符号
   - 禁止连续重复字符 > 8 次
4. 记录 ipHash 和 userAgent

## 排行榜算法

```
score = 平均分 * 20 + 收藏数 * 2 + 短评数 * 3
```

只统计 approved reviews，limit 20。

## memberUntil 字段名修复说明

### 问题
Prisma 7 默认不对 camelCase 字段名做 snake_case 映射。migration 文件最初使用了 `member_until`（snake_case），但 schema 定义的是 `memberUntil`（camelCase），导致运行时 `PrismaClientKnownRequestError: ColumnNotFound`。

### 修复
1. migration SQL 修改为：`ALTER TABLE "users" ADD COLUMN "memberUntil" TIMESTAMP(3);`
2. 生产库已执行：`ALTER TABLE users RENAME COLUMN member_until TO "memberUntil";`
3. 所有查询脚本使用带引号的 `"memberUntil"` 避免大小写问题
4. downgrade 脚本使用 `PGPASSWORD` + `psql -h 127.0.0.1` 方式连接（避免 URI 参数解析问题）

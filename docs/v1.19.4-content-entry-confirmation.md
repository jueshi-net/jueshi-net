# v1.19.4 内容入口确认 + 后台清晰化补丁

## 目标

不做大功能，只做路径确认、入口补齐、默认配置和真实健康检查。

## 修改清单

### 1. 后台首页数据来源标注

文件: `src/app/(admin)/admin/page.tsx`

- ModuleCard 接口新增 `dataSource` 字段
- 12 个模块卡片都显示数据来源（DB 表名或代码路径）
- 例如："DB: articles"、"代码: src/lib/documents"

### 2. 广告位默认配置

文件: `scripts/seed-ad-slots.ts`（新增）

Seed 5 个默认广告位：
- home-after-tools（首页工具列表下方）
- home-before-footer（首页底部之前）
- tool-bottom（工具页底部）
- sidebar（侧边栏）
- article-bottom（文章详情页底部）

每个默认 `isActive=false`，admin 需要时手动开启。

### 3. 广告管理 positionOptions 更新

文件: `src/app/(admin)/admin/ads/page.tsx`

更新下拉选项，新增 5 个推荐 placement，保留兼容旧选项。

### 4. AdBanner 安全注释

文件: `src/components/cms/ad-banner.tsx`

添加 `dangerouslySetInnerHTML` 安全注释，说明 code 类型广告仅 admin 可写。

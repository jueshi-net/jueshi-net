# 后台内容来源地图 v1.19.2

> 创建日期：2026-05-16
> 目标：让老板清楚知道后台每个内容模块在哪里管理、数据从哪里来

## 一、已上线模块（可直接使用）

### 1. 用户管理
- **页面**: `/admin/users`
- **API**: `/api/admin/users`
- **数据来源**: PostgreSQL `users` 表
- **管理方式**: 后台直接管理（查看/编辑角色/积分/会员到期时间）
- **状态**: ✅ 已上线

### 2. 工具短评审核
- **页面**: `/admin/tool-reviews`
- **API**: `/api/admin/tool-reviews`
- **数据来源**: PostgreSQL `tool_reviews` 表
- **管理方式**: 后台审核（pending → approved/hidden/rejected）
- **状态**: ✅ 已上线

### 3. 文章管理（CMS）
- **页面**: `/admin/cms`
- **API**: `/api/articles`
- **数据来源**: PostgreSQL `articles` 表（当前 20 篇文章）
- **管理方式**: 后台 Markdown 编辑器，支持草稿/发布/归档
- **状态**: ✅ 已上线（v1.19.2 修复了加载问题）

### 4. 广告管理
- **页面**: `/admin/ads`
- **API**: `/api/ads`
- **数据来源**: PostgreSQL `ad_slots` 表
- **管理方式**: 后台创建/编辑/启用/停用
- **支持类型**: 图片广告、HTML/JS 代码广告、文字广告
- **状态**: ✅ 已上线（v1.19.2 扩展支持自营广告）

### 5. 资源库
- **页面**: `/admin/resources`
- **API**: `/api/admin/resources`（需要检查是否存在）
- **数据来源**: PostgreSQL `resources` 表（当前 80 条：business 20, life 20, logistics 20, templates 20）
- **管理方式**: 目前后台有页面，需确认是否有完整 CRUD
- **状态**: ⚠️ 需验证

### 6. 链接管理
- **页面**: `/admin/links`
- **API**: `/api/admin/links`
- **数据来源**: PostgreSQL `links` 表
- **管理方式**: 后台管理
- **状态**: ✅ 已上线

### 7. 分类/标签管理
- **页面**: `/admin/categories`, `/admin/tags`
- **数据来源**: PostgreSQL `categories`, `tags` 表
- **状态**: ✅ 已上线

### 8. 单据/标签工具
- **前台页面**: `/tools/label-maker`
- **数据来源**: 前端静态配置（`src/lib/labels/label-types.ts`）
- **管理方式**: **代码级管理**，不是后台 CMS
- **模板数量**: 8 种不同类型（唛头/集运入库/合箱/库位/包裹/FBA/托盘/提示）
- **状态**: ✅ 已上线（模板选择器可见，但风格差异主要是颜色变化）

### 9. 通用单据工具
- **前台页面**: `/tools/documents`
- **数据来源**: 前端静态配置（`src/lib/documents/document-types.ts`）
- **管理方式**: **代码级管理**
- **状态**: ✅ 已上线

## 二、待开发/规划中的模块

### 1. 网盘分享管理
- **当前状态**: ❌ 未实现
- **计划**: 需要新建数据模型和后台页面
- **v1.20 优先级**: 中

### 2. 首页内容管理
- **当前状态**: 部分静态数据（`src/lib/data/`）
- **计划**: 迁移到后台 CMS
- **v1.20 优先级**: 高

### 3. 场景包/工具箱管理
- **当前状态**: 静态页面（`/starter/*`）
- **计划**: 后台可配置
- **v1.20 优先级**: 中

### 4. AI 工具管理
- **当前状态**: 前端页面 + mock 模式
- **计划**: 接入真实 AI API，后台管理 prompt/额度
- **v1.20 优先级**: 高

### 5. 论坛/社区
- **当前状态**: ❌ 未实现
- **计划**: Discourse 集成方案已调研（见 `docs/community-integration-plan.md`）
- **v1.20 优先级**: 待定

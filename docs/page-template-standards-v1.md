# 页面模板标准 v1 — xixiong-saas 海外百宝箱

> 版本：1.0  
> 创建日期：2026-05-16  
> 状态：待实施  

---

## 总则

1. **每个页面类型必须遵循固定模板结构**
2. **移动端优先** — 模板设计先考虑手机，再适配桌面
3. **区块层次清晰** — Hero → 内容 → 辅助信息 → 相关推荐
4. **空状态必须友好** — 每个有数据依赖的区域都有空状态

## 1. 首页模板

```
┌─────────────────────────────────────────────┐
│  Header (Logo + 导航 + 登录/注册)           │
├─────────────────────────────────────────────┤
│                                              │
│  Hero 区域                                    │
│  - 主标题："海外百宝箱 — 出海工具一站整理"   │
│  - 副标题：一句话说明核心价值                │
│  - 搜索框（可选）                            │
│  - CTA 按钮："开始使用"                      │
│  - 身份引导（可选）："选择你的身份"          │
│                                              │
├─────────────────────────────────────────────┤
│                                              │
│  核心工具展示区                               │
│  - 标题："常用工具"                          │
│  - 工具卡片网格（桌面 4 列，平板 3 列，手机  │
│    2 列或 1 列）                             │
│  - "查看全部工具"链接                        │
│                                              │
├─────────────────────────────────────────────┤
│                                              │
│  内容推荐区                                   │
│  - 标题："最新指南" / "推荐阅读"             │
│  - 文章卡片（桌面 3 列，手机 1-2 列）        │
│  - "查看更多文章"链接                        │
│                                              │
├─────────────────────────────────────────────┤
│                                              │
│  资源库入口                                   │
│  - 标题："更多资源"                          │
│  - 分类卡片（桌面 3-4 列，手机 1-2 列）      │
│  - "进入资源库"链接                          │
│                                              │
├─────────────────────────────────────────────┤
│                                              │
│  广告位（home-after-tools）                  │
│  - 仅当有激活广告时显示                      │
│                                              │
├─────────────────────────────────────────────┤
│                                              │
│  数据统计/信任背书（可选）                    │
│  - "X 个工具" "X 篇指南" "X 条资源"          │
│                                              │
├─────────────────────────────────────────────┤
│                                              │
│  Footer                                     │
│  - 网站信息、链接、版权                      │
│                                              │
└─────────────────────────────────────────────┘
```

### 移动端适配

- Hero 区：标题 24-28px，按钮全宽
- 工具网格：手机 1 列（大卡片）或 2 列（紧凑）
- 文章：1 列，封面图 + 标题 + 摘要
- 资源分类：2 列，图标 + 名称
- 所有卡片间距：移动端 `gap-4`，桌面端 `gap-6`

```
│                                              │
├─────────────────────────────────────────────┤
│                                              │

│  ┌───────────────────────────────────────┐  │
│  │ 输入区域                              │  │
│  │ - 标签                                │  │
│  │ - 输入框/文本域/下拉/复选框           │  │
│  │ - 帮助文字                            │  │
│  │ - [执行按钮]                          │  │
│  └───────────────────────────────────────┘  │
│                                              │
│  ┌───────────────────────────────────────┐  │
│  │ 输出区域                              │  │
│  │ - 结果展示（表格/文本/下载）          │  │
│  │ - 复制/下载按钮                       │  │
│  └───────────────────────────────────────┘  │
│                                              │
├─────────────────────────────────────────────┤
│                                              │
│  示例展示区（可选）                           │
│  - 标题："示例"                              │
│  - 输入示例 → 输出示例                       │
│                                              │
├─────────────────────────────────────────────┤
│                                              │
│  相关工具推荐                                 │
│  - 标题："你可能还需要"                      │
│  - 3-4 个相关工具卡片                        │
│                                              │
├─────────────────────────────────────────────┤
│                                              │
│  广告位（tool-bottom）                       │
│  - 仅当有激活广告时显示                      │
│                                              │
├─────────────────────────────────────────────┤
│                                              │
│  Footer                                     │
│                                              │
└─────────────────────────────────────────────┘
```

### 工具页统一规则

1. **所有工具页必须包含**：标题区、说明区、操作区
2. **操作区输入和输出必须视觉区分**（背景色或边框）
3. **按钮必须有加载状态**
4. **错误状态必须有友好提示**
5. **移动端输入区域不溢出屏幕**

## 3. AI 工具页模板

```
┌─────────────────────────────────────────────┐
│  Header                                     │
├─────────────────────────────────────────────┤
│                                              │
│  工具标题区                                   │
│  - AI 图标 + 工具名称（H1）                  │
│  - 一句话说明                                │
│  - 额度信息（今日剩余 X 次）                 │
│                                              │
├─────────────────────────────────────────────┤
│                                              │
│  输入区域                                    │
│  ┌───────────────────────────────────────┐  │
│  │ 表单字段                              │  │
│  │ [生成按钮]                            │  │
│  │ 额度提示："今日免费 X 次"             │  │
│  └───────────────────────────────────────┘  │
│                                              │
├─────────────────────────────────────────────┤
│                                              │
│  输出区域                                    │
│  ┌───────────────────────────────────────┐  │
│  │ AI 生成结果                           │  │
│  │ [复制] [重新生成]                     │  │
│  └───────────────────────────────────────┘  │
│                                              │
├─────────────────────────────────────────────┤
│                                              │
│  使用说明                                    │
│  - 这个 AI 工具能做什么                      │
│  - 使用限制（次数、字数）                    │
│  - 隐私声明                                  │
│                                              │
├─────────────────────────────────────────────┤
│                                              │
│  Footer                                     │
│                                              │
└─────────────────────────────────────────────┘
```

### AI 工具页特殊要求

1. **必须显示剩余次数/额度**
2. **加载状态必须有动画**（AI 生成需要时间）
3. **错误状态友好**（额度不足、AI 服务异常等）
4. **输出结果可复制**
5. **Mock 模式需标注**（开发阶段）

### 后台通用规则

1. **每个页面头部必须显示数据来源**
2. **表格支持搜索和筛选**
3. **批量选择 + 批量操作**
4. **空状态友好**
5. **操作有确认弹窗**（删除、状态变更）

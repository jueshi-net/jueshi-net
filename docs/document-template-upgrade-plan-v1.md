## 当前问题

### 1. 8 种模板类型只有字段差异，没有布局差异

当前所有模板使用相同的渲染逻辑，只是标签名称不同。实际上不同类型应该有不同的布局：

| 类型 | 当前状态 | 应有布局 |
|------|---------|---------|
| 通用外箱唛头 | 仅换标签 | 标准外箱格式：主唛 + 侧唛 + 重量信息 |
| 集运入库标签 | 仅换标签 | 简洁标签格式：账号 + 单号 + 目的地 |
| 集运合箱标签 | 仅换标签 | 多包裹合并格式：合箱编号 + 原包裹数 + 总重 |
| 仓库库位标签 | 仅换标签 | 小尺寸货架标签：库位码 + 区域 |
| 国际包裹面单 | 仅换标签 | 运单格式：寄件人 + 收件人 + 地址 |
| FBA 外箱信息贴 | 仅换标签 | Amazon 格式：Shipment ID + Box No + SKU |
| 托盘标签 | 仅换标签 | 大尺寸格式：Pallet No + 箱数 + 毛重 |
| 提示标签 | 仅换标签 | 警示格式：图标 + 中英文提示 |

### 2. 6 种视觉风格主要是颜色差异

| 风格 | 当前差异 | 应有差异 |
|------|---------|---------|
| 黑白 | 黑字白底 | 保持，适合打印 |
| 蓝色外贸 | 蓝色边框 | 增加企业 Logo 区域、公司信息栏 |
| 深灰仓储 | 灰色背景 | 增加条码/二维码区域 |
| 黄色警示 | 黄色高亮 | 增加警示图标、粗边框 |
| 极简 | 细线 | 减少装饰，突出内容 |
| 大字 | 大字号 | 适合远距离识别（仓库场景） |

### 3. 打印效果未优化

- 没有 `@media print` 样式
- 打印时背景色丢失
- 尺寸不匹配 A4 纸
- 没有分页控制

## 升级方案

### 阶段 1：真实布局差异

为每种类型创建独立的渲染组件：

```
src/lib/documents/
├── templates/
│   ├── shipping-mark.tsx      # 通用外箱唛头
│   ├── consolidation-inbound.tsx  # 集运入库标签
│   ├── consolidation-combined.tsx # 集运合箱标签
│   ├── warehouse-location.tsx # 仓库库位标签
│   ├── parcel-info.tsx        # 国际包裹面单
│   ├── fba-carton.tsx         # FBA 外箱信息贴
│   ├── pallet.tsx             # 托盘标签
│   └── handling.tsx           # 提示标签
├── styles/
│   ├── black-white.ts
│   ├── blue-trade.ts
│   ├── dark-warehouse.ts
│   ├── yellow-warning.ts
│   ├── minimal.ts
│   └── large-text.ts
└── index.ts
```

### 阶段 3：打印优化

```css
@media print {
  /* 隐藏非打印元素 */
  .no-print { display: none !important; }
  
  /* 确保背景色打印 */
  * { -webkit-print-color-adjust: exact !important; print-color-adjust: exact !important; }
  
  /* A4 尺寸 */
  @page { size: A4; margin: 10mm; }
  
  /* 防止分页 */
  .no-break { page-break-inside: avoid; }
  
  /* 确保尺寸准确 */
  body { width: 210mm; }
}
```

### Admin 测试说明

测试中创建 test-review-admin@local.test (role=admin in DB)，但 API 返回 role="user"。原因：NextAuth 的 JWT token 在登录时缓存了角色，session 中的 role 可能与 DB 不一致。`getCurrentUserRole` 函数从 DB 读取角色但可能受 Prisma 连接池/缓存影响。

**实际生产 admin (admin@kjbxb.com)** 不受此影响，因为该账号从未变更过角色。
